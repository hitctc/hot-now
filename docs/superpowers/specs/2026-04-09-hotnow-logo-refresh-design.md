# HotNow Logo Refresh Design

## 1. 目标

基于已经生成过的一版 `HotNow` logo，重新设计一版更符合当前偏好的主图标。

这次不是微调旧方案，而是明确替换掉上一版的主视觉方向。

新目标：

- 颜色从蓝色科技感切到 `黑白灰`
- 主要图形改成 `多条叠加的横向声波/电波线`
- 图形必须 `居中`
- 保留 `圆角方形底板`
- 继续适配 app 图标场景

## 2. 已确认方向

### 2.1 图形结构

logo 仍然是 `纯图标`，不带字标。

整体结构固定为：

- 外层：圆角方形底板
- 中间：居中的抽象雷达 / 信号图标

### 2.2 语义方向

这版强调 `雷达 / 信号感知`，图形骨架采用“多条叠加的横向声波/电波线”来承载：

- 雷达感知
- 热点扫描
- 热点感知

最终观感要更像“抽象雷达图标”，完全不能让人联想到眼睛。

### 2.3 居中要求

图形必须：

- 水平居中
- 垂直居中

不允许像上一版那样偏上放置。

## 3. 配色规范

### 3.1 主配色

采用 `黑白灰` 三层关系：

- 底板：中灰
- 主图形：白或浅灰
- 中心瞳孔 / 热点：深灰或黑

### 3.2 明确限制

这次不允许：

- 蓝色
- 青色
- 彩色高亮
- 发光感
- 霓虹科技色

目标是：

- 简洁
- 克制
- 耐看
- 更像正式品牌图标

## 4. 推荐方案

本轮采用单一推荐方案：

`圆角方形底板 + 居中多条叠加细波形`

结构建议：

- 一个中灰色圆角方形底板
- 一组居中的横向抽象波形
- 若干条叠加的细波形线条
- 波形横向展开，并在中部形成主要起伏
- 用非常克制的方式保留雷达 / 信号扫描感
- 不允许重新变回复杂多层波纹 logo
- 不允许出现圆环、圆弧雷达圈或任何像眼睛轮廓的包裹结构
- 不允许出现中心圆点、靶心或瞳孔感图形

## 5. 图形约束

必须满足：

- 一眼能看出“有雷达/信号感”
- 小尺寸下仍然清楚
- 不会误读成闹钟、闪电或 Wi-Fi
- 完全不会让人联想到眼睛
- 主图形应明显由多条横向声波/电波线构成，而不是圆形环或中心圆点构成
- 不复杂
- 不花
- 不依赖文字说明

不允许出现：

- 多层复杂波纹
- 偏上布局
- 明显科技霓虹效果
- 细碎描边和脆弱小细节

## 6. 输出规格

本轮仍然只交付两张 PNG：

- `hotnow-logo-hd.png`
  - `1024x1024`
- `hotnow-logo-sd.png`
  - `256x256`

默认沿用当前品牌资源目录：

- `src/server/public/brand/`

## 7. 与上一版的关系

上一版的主要问题是：

- 颜色不是当前想要的黑白灰
- 主要图形不符合这次想要的“多条叠加横向细波形”抽象雷达感
- 主元素视觉重心偏上，不够居中

新版要明确修复这三个问题，而不是只做轻微配色替换。

## 8. 验收标准

生成完成后，至少满足：

- 高清版尺寸正确：`1024x1024`
- 普清版尺寸正确：`256x256`
- 两张图都采用黑白灰体系
- 图形主元素明显居中
- 图标明显更像抽象雷达 / 信号感知图标，而不是普通眼睛图形
- 图形结构体现为“多条叠加横向细波形”
- 整体观感简洁
