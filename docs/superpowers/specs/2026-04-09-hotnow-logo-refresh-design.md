# HotNow Logo Refresh Design

## 1. 目标

基于已经生成过的一版 `HotNow` logo，重新设计一版更符合当前偏好的主图标。

这次不是微调旧方案，而是明确替换掉上一版的主视觉方向。

新目标：

- 颜色从蓝色科技感切到 `黑白灰`
- 主要图形从偏热点波纹切到 `抽象眼形`
- 图形必须 `居中`
- 保留 `圆角方形底板`
- 继续适配 app 图标场景

## 2. 已确认方向

### 2.1 图形结构

logo 仍然是 `纯图标`，不带字标。

整体结构固定为：

- 外层：圆角方形底板
- 中间：居中的抽象眼形信号图标

### 2.2 语义方向

这版不再强调“雷达/波纹”本身，而是强调：

- 观察
- 洞察
- 热点感知

最终观感要更像“眼睛”，而不是“雷达”。

### 2.3 居中要求

图形必须：

- 水平居中
- 垂直居中

不允许像上一版那样偏上放置。

## 3. 配色规范

### 3.1 主配色

采用 `黑白灰` 三层关系：

- 底板：中灰
- 主图形：白或浅灰
- 中心瞳孔 / 热点：深灰或黑

### 3.2 明确限制

这次不允许：

- 蓝色
- 青色
- 彩色高亮
- 发光感
- 霓虹科技色

目标是：

- 简洁
- 克制
- 耐看
- 更像正式品牌图标

## 4. 推荐方案

本轮采用单一推荐方案：

`圆角方形底板 + 居中抽象眼形 + 中心瞳孔热点`

结构建议：

- 一个中灰色圆角方形底板
- 一枚横向展开的眼形主体
- 中心一个清晰的圆形瞳孔 / 热点
- 若保留信号感，也只能非常轻，不能重新变成多层波纹 logo

## 5. 图形约束

必须满足：

- 一眼能看出“像个眼睛”
- 小尺寸下仍然清楚
- 不会误读成雷达、闹钟、闪电或 Wi-Fi
- 不复杂
- 不花
- 不依赖文字说明

不允许出现：

- 多层复杂波纹
- 偏上布局
- 明显科技霓虹效果
- 细碎描边和脆弱小细节

## 6. 输出规格

本轮仍然只交付两张 PNG：

- `hotnow-logo-hd.png`
  - `1024x1024`
- `hotnow-logo-sd.png`
  - `256x256`

默认沿用当前品牌资源目录：

- `src/server/public/brand/`

## 7. 与上一版的关系

上一版的主要问题是：

- 颜色不是当前想要的黑白灰
- 主要图形偏信号波纹，不够像眼睛
- 主元素视觉重心偏上，不够居中

新版要明确修复这三个问题，而不是只做轻微配色替换。

## 8. 验收标准

生成完成后，至少满足：

- 高清版尺寸正确：`1024x1024`
- 普清版尺寸正确：`256x256`
- 两张图都采用黑白灰体系
- 图形主元素明显居中
- 图标明显更像抽象眼睛，而不是雷达或信号波纹
- 整体观感简洁
